I'm a **FiveM developer** focused on building immersive in-game experiences through clean scripts and clean UI.
```
Frontend  →  HTML · CSS · JavaScript · NUI (FiveM)
Backend   →  Lua · Python · Node.js
```

---

## 🎨 Frontend

What I build on the client side:

- **NUI interfaces** — in-game menus, HUDs, inventory systems, notifications
- Clean and responsive layouts built with **HTML + CSS**
- Dynamic UI interactions powered by **JavaScript**
- Communication between NUI and game client via **Lua callbacks**

---

## ⚙️ Backend

What runs under the hood:

- **FiveM server-side scripting** in Lua — events, exports, player data
- **Python** for tooling, automation and utilities
- Structuring resources cleanly with clear separation of client / server / UI
